# クラウドコンピューティング

インフラストラクチャの新しい形

---

## クラウドサービスモデル

IaaS, PaaS, SaaS の違い

- **IaaS**: 仮想マシン、ストレージ、ネットワーク
- **PaaS**: アプリケーションプラットフォーム
- **SaaS**: ソフトウェアサービス
- **FaaS**: サーバーレスコンピューティング

<!-- note: サービスモデルによって責任分界点が異なります -->

---

AWS, Azure, GCP の特徴

- Microsoft Azure
